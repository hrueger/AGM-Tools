Features

Powerful dashboard

Keep track of your disk usage, upcoming events and new features!

Role based access control

Allow certain projects, settings and abilities only to certain users.

Built-in chat

Use the chat to communicate with other users either in groups or via private messages.

Integrated calendar

Organize all events for all users in the built-in calendar.

Deliver important notifications fast

Send important messages directly to specific users. They will appear on the dashboard and via push message in the mobile app!

Project based structuring

Organize your files in projects and allow specific users to join the projects.

File Management

Upload files into projects, sort them into folders, tag them and share them!
